# ⚡ SnipeX Sovereign

The #1 Solana Trading Bot with built-in token auditing powered by SolanaGuard.

## Features
- 🛡️ Instant token security audit
- 💱 Real swaps via Jupiter DEX
- 💵 Live price data via DexScreener
- 👥 Referral system with auto payouts
- 📊 Trading stats tracking
- 💎 Live SOL price

## Commands
- Paste any token address — instant audit
- /buy <token> <sol> — execute swap
- /price <token> — get price
- /stats — your trading stats
- /referrals — referral info
- /sol — live SOL price

## Tech Stack
- Node.js
- Jupiter DEX API
- DexScreener API
- QuickNode RPC
- Telegram Bot API
- Solana Web3.js
